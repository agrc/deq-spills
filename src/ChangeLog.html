  • Added Section/Township option.

1.6.1 Apr 22, 2013

  • Fixed layout bug with the clear all fields button in IE7.

1.6.0 Apr 3, 2013

  • Added Base Map Selector widget. Redmine 23863
  • "Clear All Fields" button now clears out the map search box. Redmine 23864
  • Disabled alert box error messages. Redmine 23865

1.5.0 Mar 21, 2013

  • Fires onFind event on address match now. Redmine 23584
  • "No Match Found" text changed to red. Redmine 23585
  • Added "Clear All Fields" button. Redmine 23586
  • Fixed bug in IE8 that caused an error when trying to find a route/milepost. Redmine 23589
  • Minor visual improvements.

1.4.0 Mar 19, 2013

  • Added failover to address/route&milepost so that if they fail it tries to match a city/town or county. Redmine 23545
  • Added AGRC Web API key parameter. Redmine 23560

1.3.0 Mar 18, 2013

  • Added map search bar to allow for zooming to cities, towns or counties. Redmine https://redmine.dts.utah.gov/issues/23343
  • Confirmed that parameter hierarchy is XY -> Address/Rt Milepost -> City/Town -> County -> Zoom to State. Redmine

1.2.0 Mar 14, 2013

  • Added cityName parameter. Redmine #23339
  • Added addressStreet and addressZone init parameters. These can be used for both addresses and route/mileposts. See embed-demo.html for an example. Redmine #23341
  • Added find address controls to map widget. Redmine #23342

1.1.0 Mar 13, 2013

  • Added countyName parameter. Redmine #23340
  • Upgraded to latest ESRI JavaScript API (3.3)
  • Lots of general project clean up and polishing.
